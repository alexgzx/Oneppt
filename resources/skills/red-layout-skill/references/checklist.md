# Red-note Structural Self-check

## P0 - Not Deliverable

- Content exceeds the note canvas or is clipped.
- Text-bearing modules use absolute/fixed positioning instead of grid/flex flow.
- Body text is below 18px, headings below 24px, or auxiliary text below 12px.
- The page lacks a clear hook.
- The value body is not readable as a saveable note.

## P1 - Should Fix

- The hook does not give a clear stop reason.
- The body has too many ungrouped points.
- The note lacks a bottom takeaway, action, summary, warning, or source.
- A checklist/step guide uses paragraphs instead of concise rows.
- A data note lacks nearby interpretation.
- A template note hides the reusable structure in prose.

## P2 - Consider Optimizing

- The hook could be more concrete as a benefit, question, number, or claim.
- Rows can be grouped under 2-4 section labels.
- A framework can use fewer facets with stronger labels.
- Source/footer text could be simplified into one auxiliary line.
